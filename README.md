# AI-Powered Deforestation Detection System

An AI-powered web application developed using Django and Machine Learning for detecting deforestation through satellite image analysis. The system helps monitor forest regions, classify land types, identify environmental threats, and generate analytical reports.

---

## Features

- User Authentication and Role-Based Access
- Satellite Image Upload and Processing
- AI-Based Forest Cover Classification
- Deforestation Detection System
- Land Type Identification
- Region and Town Management
- Environmental Monitoring Dashboard
- PDF Report Generation
- Image Analysis using Machine Learning
- Admin, Analyst, and Research Dashboards

---
